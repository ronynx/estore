## estore
## Installation Instructions

1. Clone the project:

       git clone https://github.com/ronynx/estore.git

2. Move project to your web server:
   - For XAMPP: copy the project folder into `C:\xampp\htdocs\`

3. Run setup.php:
   - Open in browser: http://localhost/estore/setup.php
   - This will create the database, tables, insert default admin and sample products, then automatically delete setup.php.

4. Default admin credentials:
   - Username: admin
   - Password: admin123
   - Email: admin@example.com

5. Access the application:
   - Homepage: http://localhost/estore/index.php
   - Register: http://localhost/estore/register.php
   - Login: http://localhost/estore/login.php
   - Dashboard: accessible after login
